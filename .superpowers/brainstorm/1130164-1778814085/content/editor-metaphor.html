A. Fretboard-first

Fretboard dominates. Click cells to append notes (extends today's programming mode). A thin strip below shows what's been built but isn't the primary edit surface. Duration/articulation editing happens via a context popover on each cell.

B. Timeline-first (DAW / piano-roll)

Timeline is the dominant surface. Each string is a horizontal lane; notes are bars whose length encodes duration. Drag to resize. Fretboard becomes a compact input device (or a drawer). Closest to DAW / MuseScore mental model.

C. Hybrid (equal weight)

Fretboard and timeline stacked, both primary. Fretboard for "what note goes here," timeline for "how long does it ring and what comes after." Two coordinated views of the same pattern.

What stays the same in all three: the data model (timeline of timed events, optionally lane-tagged), the keyboard shortcuts/transport, and the relationship to the metronome. The difference is which surface the user spends their time looking at while authoring.
